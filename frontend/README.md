# Aivoa CRM - Frontend UI Client

This directory contains the React (Vite) frontend application for the **Aivoa CRM HCP Module**.

For the full project documentation including system architecture diagrams, database schema, data flow sequences, and full-stack setup instructions, please see the main [root README.md](file:///c:/aivoa/README.md).

## Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the local development server:
   ```bash
   npm run dev
   ```

3. Open `http://localhost:5173` in your browser.

*Note: Ensure the backend FastAPI server is running on port 8000 for full features (chat agent, database logging, voice summarization).*
